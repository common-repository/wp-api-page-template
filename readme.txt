Extends WordPress WP REST API page endpoints with page template name.

== Description ==
Extends [WordPress JSON REST API](https://wordpress.org/plugins/rest-api/) page endpoints with page template name.

== Installation ==
This plugin requires having [WP API](https://wordpress.org/plugins/rest-api/) installed and activated or it won\'t be of any use.

Install the plugin as you would with any WordPress plugin in your `wp-content/plugins/` directory or equivalent.

Once installed, activate WP API Page template from WordPress plugins dashboard page and you\'re ready to go, WP API /pages resonse now includes template endpoint.
